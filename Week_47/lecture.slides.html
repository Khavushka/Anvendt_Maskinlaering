Preliminaries (2): Attention [1]¶

Transformers use a mechanism known as attention

This controls which part(s) of the network's inputs it should focus on ("pay attention to")

Why does this matter? Ofen not a one-to-one mapping between input and output

Examples:

  1. Translation from one language to another (word order)
  2. Summarization (content order)
  3. Script generation (idea order)

What is a Transformer? [2]¶

Take a NN. Remove everything but attention. You now have a transformer (possibly slightly oversimplified)

Query-key-value approach: $\texttt{Attention}(Q, K, V) = \texttt{softmax}(\frac{QK^T}{\sqrt{d_k}})V$

In [119]:
import math, torch

def attention(query, key, value) -> torch.Tensor:
    d_k = query.size(-1)
    scores = torch.matmul(query, key.transpose(-2, -1)) / math.sqrt(d_k)
    p_attn = scores.softmax(dim=-1)

    return torch.matmul(p_attn, value)

How to Train a Transformer¶

Transformers benefit tremendously from scale and large datasets

SOTA performance generally follows the pattern (1) pre-train on a large dataset and then (2) fine-tune on the desired task

Question: How do we (pre)-train a transformer? Specifically, which objective to optimize?

Note: We ultimately want versatile models: They should be able to translate, proof-read, help us code, summarize, etc.

Idea: Next-word prediction, e.g., "The cat sat on the __" ← now fill in the next word

Limitaton: Network only attends to previous words

  • Useful for generative models (write me a program to do ...)
  • Potentially harmful for word-level tasks such as Q&A, where context from the right might also be crucial

BERT: Bidirectional Encoder Representations from Transformers [3]¶

Idea: Train transformers on Cloze tests

Cloze test: Context and vocabulary test described by Wilson L. Taylor in 1953

How many are familiar with the Cloze test?

BERT: Training [3]¶

Cloze test: "The __ sat on the mat" now fill in the <BLANK> (this is slightly simplified)

Result: Able to pre-train transformers that can attend to everything, not just to previous elements

What was wild about BERT?: It achieved SOTA on 11 benchmarks within NLP with a unified architecture across tasks

Note: This is not always optimal. For example, limited use for generative tasks

import torch

class PatchEmbed(torch.nn.Module):
    def __init__(self, patch_size=(16, 16)):
        super().__init__()
        self.proj = nn.Conv2d(
            3, 768, kernel_size=patch_size, stride=patch_size)
        
    def forward(self, x):
        x = self.proj(x)
        x = x.flatten(2).transpose(1, 2)  # NCHW -> NLC
        
        return x

class Attention(torch.nn.Module):
    def __init__(self):
        super().__init__()

        self.head_dim = 768 // 8 # 8 heads
        self.scale = 8 ** -0.5 # this is d_k

        self.qkv = nn.Linear(768, 768 * 3)
        self.proj = nn.Linear(768, 768)

    def forward(self, x):
        B, N, C = x.shape

        q, k, v = self.qkv(x).reshape(B, N, 3, 8, self.head_dim).permute(2, 0, 3, 1, 4).unbind(0)
        q = q * self.scale
        
        attn = q @ k.transpose(-2, -1).softmax(dim=-1) # recall formula
        x = attn @ v

        x = x.transpose(1, 2).reshape(B, N, C)
        x = self.proj(x)
        
        return x

class VisionTransformer(torch.nn.Module):
    def __init__(self):
        super().__init__()
        self.patch_embed = PatchEmbed()
        self.attn = Attention()
        self.head = torch.nn.Linear(768, 3)
    
    def forward(self, x):
        x = self.patch_embed(x)
        # Here we would add some positional info
        x = x + self.attn(x)
        x = self.head(x.mean(dim=1))
        
        return x

from timmsn.models.layers import TransformerDecoder

class ImageToTextTransformer(nn.Module):
    def __init__(self):
        super().__init__()
        self.encoder = VisionTransformer() # ViT
        self.decoder = TransformerDecoder(3, 768, 8, 30) # Transformer

    def encode(self, image): # encode image as sequence
        return self.encoder.forward_features(image)

    def decode(self, memory, target_mask, target_padding_mask):
        return self.decoder(memory, target, target_mask, target_padding_mask)

    def forward(self, image, target, target_mask, target_padding_mask):
        memory = self.encode(image)
        out = self.decode(memory, target, target_mask, target_padding_mask)

        return out
